## 自己学習 (候補)

- **Playwright/Vite の落とし穴**: `vite preview` 配信の E2E は `.env.local` の `VITE_*` を build に inline するため、env 依存の分岐 (keyless graceful 等) を検証するテストは ambient キーで揺れる。**no-key スイートの webServer ビルドでは検証対象の `VITE_*` を空に固定**して決定性を担保する (process.env が .env ファイルより最優先)。`/flow:e2e` の自己学習に昇華候補。
