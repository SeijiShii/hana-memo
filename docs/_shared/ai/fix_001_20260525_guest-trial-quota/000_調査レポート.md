## 1. 症状
新規匿名(ゲスト)ユーザーが画像を 1 回選んだ時点で「今月の識別回数を使い切りました (残り 0)」モーダルが出て、AI 同定が一度も実行できない。

## 2. 期待挙動
新規匿名ユーザーは無料枠 (ANON_TRIAL_MAX=3 回) で AI 同定できる (concept §4 / ai SPEC §4 L100「ai_credits_remaining or trial」)。

## 3. 再現手順
1. 新規起動 (匿名サインイン成立、users.trial_used_count=0, ai_credits_remaining=0)
2. 撮影で画像を選択
3. → 撮影前 quota ゲートで即「使い切り」モーダル

## 4. 再現環境
localhost:3000 (vercel dev) / Windows ブラウザ / 実 Clerk test + 実 Neon。全新規ユーザーで決定的に再現。

## 5. 影響範囲
- 影響ユーザー: 全新規ユーザー (匿名スタート全員)。無料枠が 0 回も成立しない
- 影響機能: コア体験 (撮るだけで名前) + それに連なる保存・図鑑
- データ影響: なし / セキュリティ影響: なし
