# プライバシーポリシー

**バージョン**: v1.1.0
**最終更新**: 2026-05-24

## 1. はじめに

本ポリシーは、植物識別 PWA「hana-memo」(以下「本サービス」) の提供者 (以下「当方」) が、本サービスの利用にあたって取得・利用する個人情報の取扱いについて定めるものです。

## 2. 取得する情報

本サービスは以下の情報を取得します。

- **匿名識別子**: Supabase Auth による匿名 user ID (UUID)。デバイス／ブラウザに紐づきます。
- **撮影画像**: ユーザーが撮影し本サービスに保存する植物写真。Supabase Storage に保存されます。
- **位置情報** (任意): 撮影時の緯度・経度。**約 100m 単位に丸め** (デフォルト)、精細位置はサーバーに送信されません。設定で完全 OFF も可能。
- **同意ログ**: 法的同意の記録 (doc_type / doc_version / 同意日時 / IP のハッシュ値)。
- **API 利用ログ**: AI 識別利用回数、課金履歴、エラーログ。
- **アカウント連携情報** (任意): Google OAuth でアカウント連携した場合、Google から提供されるメールアドレス・表示名・プロフィール画像 URL。
- **エラーレポート** (opt-in): Sentry によるクライアントエラー情報。デフォルト OFF。

## 3. 利用目的

取得した情報は以下の目的でのみ利用します。

- 本サービスの提供 (画像保存・AI 識別・観察ノート機能)
- 不具合対応・改善 (opt-in user のみ)
- 月次利用枠の管理・課金処理
- 法令遵守 (個人情報保護法、特定商取引法等)

## 4. 第三者提供

以下の場合を除き、ユーザーの同意なく第三者に個人情報を提供しません。

- **AI 識別委託**: 撮影画像を OpenAI 社 (米国) の Vision API に送信します。OpenAI の利用規約に基づき、API 経由のデータは原則として学習に使用されません ([OpenAI API Data Usage Policy](https://openai.com/policies/api-data-usage-policies))。
- **画像保存**: Supabase Inc. (米国) のホスティングサービスを利用します。
- **エラー監視 (opt-in)**: Sentry 社 (米国、エラー追跡委託先) にエラーレポートを送信します。送信前に、メールアドレス・位置情報 (緯度経度)・決済識別子・セッショントークン等の個人情報を当方のクライアント / サーバーでスクラブ (除去) し、個人を特定できる情報を含めずに送信します。本機能は opt-in (デフォルト OFF) で、設定からいつでも無効化できます。
- **決済処理**: Stripe Payments Japan 株式会社 を通じた決済時に、必要な情報を Stripe に提供します。クレジットカード番号は当方サーバーを経由しません。
- **法令に基づく場合**

## 5. 越境移転

本サービスは米国のクラウドサービス (OpenAI / Supabase / Sentry / Vercel / Stripe) を利用するため、ユーザーの個人情報は米国に移転されます。米国の個人情報保護制度については個人情報保護委員会のガイダンスを参照ください。

## 6. データ保存期間

- **画像・観察ノート**: アカウント削除まで保持。
- **同意ログ**: 法的トレース性確保のため、アカウント削除後も最大 5 年間保持。
- **IP ハッシュ**: 90 日後に削除。
- **API 利用ログ**: 集計用にハッシュ化された形で 13 ヶ月保持。

## 7. ユーザーの権利

ユーザーは以下を請求できます。

- **開示・訂正・削除**: 設定画面「アカウント削除」または `/legal/contact` から依頼。
- **AI 識別の利用停止**: 設定画面「AI 利用同意」を OFF にすることで以後の AI 識別を停止 (過去ログは保持)。
- **エラーレポート送信停止**: 設定画面「品質改善への協力」を OFF。

## 8. Cookie 等の利用

本サービスは MVP 段階では analytics 系 Cookie を使用しません。Supabase Auth のセッショントークン (cookie / localStorage) のみ使用します。将来的に Cookie ポリシーを別途定める場合があります。

## 9. 子どもの利用

13 歳未満のユーザーは、保護者の同意のもとで利用してください。13 歳未満からの個人情報取得が判明した場合、当方は速やかに削除します。

## 10. 改訂

本ポリシーは予告なく改訂することがあります。重要な変更がある場合は、本サービス起動時に再同意ダイアログを表示します。
