## 1. 変更概要

§8 [論点-014] (SEC-004、High / 法令必須) の closure 法務 TODO に基づき、プライバシーポリシー §4「第三者提供 — エラー監視 (opt-in)」に、Sentry へ送信する**前に**個人情報をスクラブ (除去) する旨を明記する。SEC-004 実装 (`_shared/analytics/scrubber.ts` の 7 パターン scrub + Sentry `beforeSend`/`beforeBreadcrumb`) を開示し、委託先 (Sentry) への PII 流出ゼロを利用者に保証する。プラポリのバージョンを v1.0.0 → v1.1.0 に更新。

### [論点-001] version bump で再同意を強制するか
