# AI 利用に関する同意事項

**バージョン**: v1.0.0
**最終更新**: 2026-05-22

---

## 1. 利用する AI サービス

本サービスは植物識別機能において、**OpenAI 社の Vision API (gpt-4o-mini)** を利用します。

## 2. 送信されるデータ

ユーザーが撮影・アップロードした **植物画像** が OpenAI API に送信されます。同時に、識別精度向上のため以下の付帯情報も送信される場合があります。

- 撮影日時 (秒精度)
- 季節情報 (春／夏／秋／冬)
- 大まかな位置情報 (約 100m 単位、デフォルト ON、設定で OFF 可)
- ユーザーが入力した補助メモ (任意)

ユーザー識別子 (UUID) や端末固有情報は送信されません。

## 3. OpenAI 社のデータ取扱い

OpenAI の API 経由で送信されたデータは、[OpenAI API Data Usage Policy](https://openai.com/policies/api-data-usage-policies) に基づき、**原則として AI モデルの学習に使用されません**。同社の保存期間は通常 30 日以内です (悪用検知目的)。

## 4. 識別結果の性質

- AI による識別結果は **参考情報** であり、当方および OpenAI は識別の正確性を保証しません。
- 有害植物 (毒性植物・刺激性植物等) を誤って安全と判定する可能性があります。
- 識別結果を基に植物に触れる・摂取する・利用する場合は、必ず複数の信頼できる情報源 (図鑑・専門家) で確認してください。

## 5. 利用回数制限

- 無料枠: 月 10 回まで AI 識別が可能です。
- 超過後は ¥100 / 20 回の追加クレジットを購入できます。

## 6. 同意の撤回

ユーザーは設定画面の「AI 利用同意」を OFF にすることで、以後の AI 識別を停止できます。停止後も過去の識別結果ログ (api_usage テーブル) は保持されます (請求時に削除可)。

## 7. 改訂

OpenAI のポリシー変更や利用モデルの変更があった場合、本同意事項を改訂し、起動時に再同意ダイアログを表示します。
